安装配置

Houdini

Maya

在

指定config.ocio的路径并启用Use OCIO Configuration

Nuke

  • 临时设置
    S键调出工程设置，在Color栏设置Color Management为OCIO
    就可以在项目中使用opencolorio

  • 永久设置
    Edit > Preferences > Project Defaults > Color Managment
    选择aces色彩空间点击OK保存即可
